## 🔧 Installation

If you don't have rust installed, check the [Rust website](https://rust-lang.org/tools/install/).

```shell
git clone https://github.com/LeoLetellier/RasterView
cd RasterView
cargo build --release
```

The binary will be located in `./target/release/`.
